# کاروان اسلام

### البعثه الاسلامیه الیبلاد الافرنجیه

اثر: **صادق هدایت** (۱۳۱۲–۱۳۱۳ هجری شمسی)

طنزنامه‌ای کوتاه در قالب سه نامه از خبرنگار مجله «المنجلاب» که همراه کاروان بعثه الاسلامیه به سوی بلاد فرنگ روانه شده و گزارش روزانه آن را می‌نویسد.

---

## درباره این نسخه

جمعی از علاقه‌مندان ادبیات معاصر فارسی این اثر را در قالب کتاب الکترونیکی (mdBook) بازنشر کردند. ویژگی‌های این نسخه:

- پایبندی کامل به متن در دسترس
- تصاویر اصلی کتاب به شکل مجزا
- ضمیمه توضیح واژه‌ها و اصطلاحات
- یادداشت‌های حاشیه‌ای و مقدمه ناشر

## ساختار کتاب

| بخش | شرح |
|---|---|
| پیشگفتار | مقدمه انتشار اینترنتی |
| دیباچه | معرفی اثر و نویسنده |
| نامه اول | جلسه اعزام مبلغین در سامرا |
| نامه دوم | ورود به برلین و نمایشگاه شرقی |
| نامه سوم | سه سال بعد در پاریس — فروشگاه میسر |
| حاشیه | یادداشت‌ها و حواشی نشر کتاب |
| تصاویر | طرح‌های خود صادق هدایت |
| افسانه آفرینش | نمایشنامه خیمه‌شب‌بازی در سه پرده |

## ساخت و اجرا

پیش‌نیاز: [mdBook](https://github.com/rust-lang/mdBook)

```bash
# نصب mdBook
cargo install mdbook

# ساخت کتاب
mdbook build

# پیش‌نمایش زنده
mdbook serve
```

خروجی در دایرکتوری `book/` تولید می‌شود.
